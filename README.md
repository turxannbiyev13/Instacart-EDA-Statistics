# Instacart-EDA-Statistics 🛒

This repository contains a comprehensive Exploratory Data Analysis (EDA) and statistical breakdown of the **Instacart open-source dataset**. The project is designed to uncover actionable insights into customer shopping behavior, order frequency, and peak ordering patterns.

## 📊 Project Overview
The primary goal of this project is to apply statistical methods—such as mean, variance, and correlation analysis—to understand the underlying trends in customer purchase history. It serves as both a practical guide for data analysis workflows and a portfolio piece for statistical modeling using Python.

## 🔑 Key Insights
* **Customer Behavior:** Analyzed return cycles to understand customer loyalty.
* **Peak Patterns:** Identified peak ordering times during the day, correlating with population behavior.
* **Statistical Correlation:** Discovered a negative correlation between order frequency and the time gap between orders, indicating increased engagement over time.

## 🛠 Tech Stack
- **Python** (Pandas, NumPy)
- **Data Visualization** (Seaborn, Matplotlib)
- **Environment** (Google Colab)
